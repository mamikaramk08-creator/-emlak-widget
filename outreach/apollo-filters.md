# Apollo.io / Apify Lead Finder — Filter Checklist

Use this exact filter set for the first lead-list pull.

- **Industry:** Real Estate
- **Title:** Owner, Broker, Broker-Owner, Managing Broker, Principal, Founder
  (small firms → the broker-owner is the buyer, no marketing-director layer)
- **Company size:** 1–50 employees
- **Location:** United States — start with 1–2 high-density metro/state
  filters (e.g. Texas, Florida) to keep the first batch narrow, then expand
  once reply-rate data comes in
- **Exclude:** corporate/franchise HQ accounts (RE/MAX corporate, Compass
  corporate, Coldwell Banker corporate, etc.) — independents only
- **Email:** verified email required

## Manual secondary pass (optional, first batch only)

Spot-check each prospect's website for an existing live-chat widget
(Intercom / Drift / Tidio / etc. script tags) and deprioritize agencies that
already have one. Do this by hand on the first batch rather than building
automation for it yet.

## Batch size

First pull: ~200–300 contacts, narrow filters above. Spot-check the first 10
results by hand (right title, plausible independent brokerage, real US
location) before pulling the full batch.
